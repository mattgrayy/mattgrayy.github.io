Pipeline Express

Core Responsibilities

- Working in a small team to fix bugs and add new features

- Designing levels and gameplay experiences with new features

Description

Pipeline Express is a single player water management mobile and web 'serious game'.

Use your limited water pipes to flow water to each building in the town, plan the layour carefully to fuel concrete factories to generate more pipes. Later levels include sewage management as well as water management. Create levels using the level editor and challenge your friends!
